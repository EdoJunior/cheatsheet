HTML/CSS CheatSheet

HTML Starter Tags

Tag: Description:
<!DOCTYPE html> defines the document type
<html> defines an HTML document
<head> contains metadata information for the document
<title> defines a title for the document
<body> defines the documents body
<div> defines a section in a document
<h1 - h6 > defines HTML headings
<p> defines a paragraph
<br> inserts a single line break
<!--...--> defines a comment
<link> defines the relationship between a document and an external resource (most used to link to style sheet)

Table Tags:

Tag: Name: Description:
<table> Table Defines a table.
<thead> Table Head Groups the header content in a table.
<tbody> Table body Groups the body content in a table.
<tr> Table Row Defines a row in a table.
<td> Table Data Defines a cell in a table.
<tfoot> Table Foot Groups the footer content in a table.
<col> Columns Specifies column properties for each column within a <colgroup> element.
<colgroup> Column Group Specifies a group of one or more columns in a table for formatting.
